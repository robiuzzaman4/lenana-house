# Lenana House

A premium, private villa website designed for seasonal living and quiet stays. This project features a sophisticated, minimal aesthetic with a focus on immersive visual experiences.

- **Components:** Completely custom UI components including sticky navigation, dynamic galleries, and animated scroll indicators.

- **Responsive Design:** Optimized for all screen sizes from mobile to ultra-wide displays.

- **Integrated Sections:** Specialized pages for Residence, Rooms, Experiences, and a full Media Gallery.
